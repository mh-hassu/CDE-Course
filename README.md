# Cloud Data Engineering

A comprehensive learning repository for Python, SQL, and Data Engineering concepts.

## Repository Structure

```
Cloud Data Engineering/
├── Python Classes/          # Python learning materials & class notes
├── SQL Classes/             # SQL learning materials & queries
├── Assignments/             # Practice assignments & exercises
├── My Practice/             # Personal practice projects & notebooks
└── pdf/                     # Reference materials & PDFs
```

---

## Python Classes

Comprehensive Python coursework covering:

| Topic | Description |
|-------|-------------|
| Variables & Operators | Basics of variables, operators, and concatenation |
| Input & Conditions | User input, conditional statements, and slicing |
| Dictionaries | Dictionary operations and methods |
| For Loops | Loop structures and iterations |
| Functions | Function definitions, parameters, and return values |
| Lambda Functions | Anonymous functions and functional programming |
| Classes & OOP | Object-oriented programming, inheritance, method overriding |
| File Handling | Working with CSV, JSON files, and error handling |
| NumPy & Matplotlib | Data manipulation and visualization |
| Web Scraping | Extracting data from websites |

## SQL Classes

SQL learning materials covering:

- **Basics**: WHERE, LIKE, TOP, ORDER BY, FETCH
- **DDL & Joins**: Data Definition Language and table joins
- **Aggregations**: GROUP BY, HAVING, Subqueries
